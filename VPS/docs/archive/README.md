# Archives

Ce dossier conserve les anciennes notes, les brouillons et les variantes retirées
de la procédure active.

La documentation active commence dans le [README principal](../../README.md).

## Règle

- Ne pas suivre directement ces fichiers pour une nouvelle installation.
- Les consulter seulement pour retrouver une décision, une ancienne variante ou
  un brouillon non intégré.
